# CP Command examples:
---

Copy a file to another location:
```bash
cp file.txt /path/to/destination/
```
This command copies the file named file.txt to the /path/to/destination/ directory.

---
```bash
cp file1.txt file2.txt file3.txt /path/to/destination/

```
